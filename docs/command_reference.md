
# Command Reference

This document contains the Linux and Python commands that were actually used during the development of the Berlin-Box.

It is intended as a quick reference and complements the installation guide. Every command listed here has been tested during the project.

---

## Raspberry Pi

Useful commands for system administration.

Examples:

* Update the operating system
* Reboot the Raspberry Pi
* Shut down the Raspberry Pi safely

---

## Python Virtual Environment

Commands related to the Python virtual environment (venv).

Examples:

* Create a virtual environment
* Activate the virtual environment
* Leave the virtual environment
* Verify the installed Python version

---

## Reticulum

Commands used to install, configure and verify the Reticulum Network Stack.

Examples:

* Check the installed version
* Verify the configuration
* Check the Reticulum service
* View Reticulum log messages

---

## LXMF

Commands related to LXMF.

Examples:

* Install LXMF
* Verify the installation
* Check the installed version

---

## Berlin-Box Service

Commands used to control the Berlin-Box system service.

Examples:

* Start the service
* Stop the service
* Restart the service
* Check the service status
* View the service log

---

## Diagnostics

Useful commands for troubleshooting.

Examples:

* Verify the connected RAK4631
* Locate the Berlin-Box script
* Check Python package locations
* Display system information

---

## Backup

Commands and procedures used to back up and restore the Raspberry Pi SD card.

---

## Notes

Only commands that have been successfully used during the development of the Berlin-Box should be added to this document.

This keeps the reference practical, reliable and easy to follow.
